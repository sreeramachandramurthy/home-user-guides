# Install FreeCAD

## Add Repository

* Open Terminal
* Run `sudo add-apt-repository ppa:freecad-maintainers/freecad-stable`
* Run `sudo apt update`

## Installation

* Open Terminal
* Run `sudo apt install freecad`

## Installation of Offline Docs

* Open Terminal
* Run `sudo apt install freecad-doc`

## Run FreeCAD

* Open Terminal
* Run `./freecad`

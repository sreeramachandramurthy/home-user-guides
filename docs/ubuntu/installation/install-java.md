# Install Java

## Installation Steps

### Install Java Common

* Open Terminal
* Run `sudp apt install java-common`

### Install OpenJDK 8

* Open Terminal
* Run `sudo apt install openjdk-8-jdk`

### Install OpenJDK 11

* Open Terminal
* Run `sudo apt install openjdk-11-jdk`

### Install OpenJDK 18

* Open Terminal
* Run `sudo apt install openjdk-18-jdk`

## Java Alternatives

* Open Terminal
* To list Java versions run `update-java-alternatives --list`

# Install Kodi

## Installation Steps

* Open Terminal
* Install dependencies for Kodi

```bash
sudo apt install software-properties-common
```

* Add Kodi repository

```bash
sudo add-apt-repository ppa:team-xbmc/ppa
```

* Install Kodi

```bash
sudo apt install kodi
```

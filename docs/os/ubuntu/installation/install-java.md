# Install Java

## Installation Steps

### Install Java Common

* Open Terminal
* Install Java Common

```bash
sudo apt install java-common
```

### Install OpenJDK 8

* Open Terminal
* Install Open JDK 8

```bash
sudo apt install openjdk-8-jdk
```

### Install OpenJDK 11

* Open Terminal
* Install Open JDK 11

```bash
sudo apt install openjdk-11-jdk
```

### Install OpenJDK 18

* Open Terminal
* Install Open JDK 18

```bash
sudo apt install openjdk-18-jdk
```

## Java Alternatives

* Open Terminal
* To list diiferent Java versions in the system, run the below command

```bash
update-java-alternatives --list
```
